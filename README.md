Мои проекты на C++



 1. Калькулятор
Простой калькулятор. Умеет складывать, вычитать, умножать, делить. Работает в цикле, пока пользователь не выйдет.

Файл: `calculator.cpp`



 2. Анализатор диапазона
Программа, которая считает сумму и количество чётных или нечётных чисел в заданном диапазоне.

Файл `range-parity-analyzer.cpp`



 3. Охота на тролля (текстовая RPG)
Моя первая игра. Полноценная текстовая RPG с тремя локациями, торговлей, игрой в кости и несколькими концовками.

Файлы: `Troll hunt.cpp`, `Troll_Hunt.exe`
